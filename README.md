# TD Operations — CRM Dashboard

Tony Durante LLC operational dashboard connected to Airtable.

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to vercel.com → New Project → Import the repo
3. Click Deploy — done

The dashboard will be live at `your-project.vercel.app`

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack
- Next.js 14 (App Router)
- Airtable API (direct fetch)
- No external UI libraries
